# Temperature Checker

## Description

Temperature Checker is a simple Python program that accepts a temperature value from the user and classifies the weather condition based on the input.

The program determines whether the temperature is:

* Freezing 🥶
* Too Cold ❄️
* Normal 😊
* Warm ☀️
* Too Hot 🔥

## Features

* User-friendly input system
* Automatic temperature classification
* Simple and easy-to-understand code
* Great project for Python beginners

## How to Run

1. Make sure Python is installed on your computer.
2. Download or clone this repository.
3. Open a terminal in the project folder.
4. Run the program:

python temperature_checker.py

5. Enter a temperature value when prompted.

## Example

Input:
12

Output:
Too Cold ❄️

## Purpose

This project was created to practice Python fundamentals such as user input, conditional statements (`if`, `elif`, `else`), and output formatting.
